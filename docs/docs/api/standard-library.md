# Standard Library

Built-in functions available in Fax.

## Print

```fax
print(value)  // Print value to stdout
```

Examples:

```fax
print("Hello");   // String
print(42);        // Number
print(true);      // Boolean
```

## Input (Future)

Input functions are planned for future releases.

## Type Conversion

Type conversion functions are planned:

- `to_string(x)` - Convert to string
- `to_i64(x)` - Convert to integer
