# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

Since this is an experimental compiler, please report vulnerabilities by opening a [GitHub Issue](https://github.com/fax-lang/faxc/issues/new) with the label `security`.

If the vulnerability is critical, please refrain from posting public exploits immediately.
