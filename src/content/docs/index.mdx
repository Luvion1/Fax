## Why Fax?

<CardGrid stagger>
  <Card title="🚀 High Performance" icon="rocket">
    Optimized compiler pipeline with Rust, Zig, and C++ components for maximum speed.
  </Card>
  <Card title="🧠 Generational GC" icon="setting">
    Custom FGC with nursery, multiple page sizes, and parallel marking for predictable memory management.
  </Card>
  <Card title="🔧 Polyglot Design" icon="puzzle">
    Each compiler stage uses the best tool - Lexer in Rust, Parser in Zig, Sema in Haskell, and Codegen in C++.
  </Card>
  <Card title="📦 Module System" icon="document">
    Modern module system with imports, exports, and dependency management.
  </Card>
  <Card title="🛡️ Memory Safety" icon="approve-check">
    Bounds checking, null safety, and comprehensive static type system.
  </Card>
  <Card title="✨ Modern Syntax" icon="star">
    Clean syntax inspired by Rust, Go, and TypeScript with pattern matching.
  </Card>
</CardGrid>

## Quick Start

```bash
# Clone the repository
git clone https://github.com/Luvion1/Fax.git
cd Fax

# Install dependencies
npm install

# Build all components
make build

# Run your first program
python3 faxt/main.py run examples/hello.fax
```

## Language Example

```fax
struct Person {
    name: str,
    age: i64
}

fn greet(person: Person) {
    print("Hello, " + person.name);
}

fn main() {
    let p = Person {
        name: "Alice",
        age: 30
    };
    
    greet(p);
}
```
