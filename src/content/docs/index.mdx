## Why Choose Fax?

Fax combines the best of modern language design with cutting-edge compiler technology. Each stage of our pipeline is crafted in the optimal language for maximum performance and reliability.

</div>

<CardGrid stagger>
  <Card title="⚡ Blazing Fast" icon="rocket">
    **Zero-cost abstractions** with LLVM optimization. Compile times measured in milliseconds, not seconds.
  </Card>
  <Card title="🧠 Smart GC" icon="setting">
    **Generational garbage collector** with nursery, parallel marking, and write barriers. Predictable pause times under 10ms.
  </Card>
  <Card title="🎯 Type Safe" icon="approve-check">
    **Static typing with inference**. Catch bugs at compile time, not runtime. Full null safety guaranteed.
  </Card>
  <Card title="🔧 Polyglot Power" icon="puzzle">
    **Best-of-breed architecture**: Rust lexer, Zig parser, Haskell type checker, C++ codegen, Zig runtime.
  </Card>
  <Card title="📦 Modern Modules" icon="document">
    **First-class package system** with semantic versioning, dependency resolution, and reproducible builds.
  </Card>
  <Card title="🛡️ Memory Safety" icon="shield-check">
    **Bounds checking, null safety, and no data races**. Write secure code without the overhead.
  </Card>
</CardGrid>

## See It in Action

<div class="code-showcase">

<Code lang="fax" title="hello.fax" 
code={`// Define a struct with named fields
struct Point {
    x: i64,
    y: i64
}

// Function with type annotations
fn distance(p1: Point, p2: Point): f64 {
    let dx = p2.x - p1.x;
    let dy = p2.y - p1.y;
    return sqrt(dx * dx + dy * dy);
}

// Pattern matching with match expressions
fn classify(value: i64): str {
    match value {
        case 0: { return "zero"; }
        case 1..10: { return "small"; }
        case 11..100: { return "medium"; }
        default: { return "large"; }
    }
}

fn main() {
    let origin = Point { x: 0, y: 0 };
    let target = Point { x: 3, y: 4 };
    
    let dist = distance(origin, target);
    print("Distance: " + dist);  // Distance: 5.0
    
    print(classify(42));  // medium
}`}/>

## Language Features

<div class="feature-grid">

### 🔥 Modern Syntax
Clean, expressive syntax inspired by the best of Rust, Go, and TypeScript.

### 🎯 Pattern Matching
Powerful `match` expressions with exhaustive checking and guards.

### 🔗 First-Class Functions
Functions are values. Pass them around, store in variables, return from functions.

### 🏗️ Zero-Cost Abstractions
High-level features compile down to efficient machine code.

## Quick Start

<div class="quick-start">

### 1. Install

```bash
# Clone the repository
git clone https://github.com/Luvion1/Fax.git
cd Fax

# Install dependencies
npm install

# Build the compiler
make build
```

### 2. Write

Create `hello.fax`:

```fax
fn main() {
    print("Hello, World!");
}
```

### 3. Run

```bash
python3 faxt/main.py run hello.fax
```
